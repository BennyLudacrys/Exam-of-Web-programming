A data de devolução não pode ser estendida

A data de devolução do livro que você está tentando estender não pode ser estendida. Possíveis razões são:

- O período máximo de uso é atingido.
- A data de devolução já foi prorrogada por 3 semanas.
- Outra pessoa reservou o livro.

Por favor, certifique-se de devolver este livro a tempo para a biblioteca.
